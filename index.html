Traffic Light Control System

In the current growing world of automobiles the road networks are getting smart. We need smart solutions for smart cities. In coming days use of smart vehicles will be more and more common. For fluid traffic and to avoid congestion on the roads so we built a smart automated and scalable approach to traffic signal system.


This project aims to minimize the overhead infrastructure development required onsite to handle modern day traffic congestion issues without the massive amount of investment. The project can be expanded upon in the future by adding more features. The use of Artificial intelligence based system such as Deep neural networks and Sequence models is that they are scalable as the data gets greater and greater. It also allows the system to take decisions for conditions which haven’t been encountered before, giving us a more reliable and fluid experience.This project aims to minimize the overhead infrastructure development required onsite to handle modern day traffic congestion issues without the massive amount of investment. Video stream input is fed into the program by the CCTV cameras. The video is decoded and made into MJPEG format if required then the video is converted into a stream of JPEG images. The JPEG images are used to as input for the CNN. One JPEG each 5 sec or 10 sec interval will be picked from the stream and will be used to run the detection over.Then this normalized value is inserted into a Deep Neural Network(DNN) along with the car count. The DNN is trained on synthetic dataset which is made to predict the decision that needs to be taken at the signal.(image of proposed model architecture on the right) The DNN’s response is recorded in the SQL database along other information. The data that is stored in the database is used to train the sequence model.The sequence model is made using LSTM or GRU.(still working on architecture and testing accuracy & cross-val scores) The LSTM provides prediction to the DNN model. The DNN model’s decision and current traffic information is displayed on the website with live updates.

Problems

Unnecessary congestion and wait times caused due to timed traffic signal systems where infrastructure like flyovers are not good investment.



Solution

To provide a reliable system to dynamically take decision and control the traffic signals at node using Deep learning to detect and predict the flow of traffic while maintaining scalability.

Our Aim

Allowing proper traffic Surveillance and traffic management system in case of emergency or any other situation..
